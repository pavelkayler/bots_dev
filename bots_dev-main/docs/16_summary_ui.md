# 16 Summary UI (post-run)

Last update: 2026-02-25

## Reset on Start
Summary clears immediately when a new session starts, so the operator does not see previous-run results during the run.

## Load on Stop
After Stop, summary is fetched and displayed.

## Trades table
- Global sorting: clicking any column header sorts across the full dataset.
- Pagination applies after sorting.
- Page size selector: 50 / 100 / 200
- Fees are displayed as negative values for clarity.
