# 74 - Signal Bot page split (tabs) and optimizer scoping

Date: 2026-03-07

## 1) Goal
Apply the updated UX flow for the new signal bot:
- keep bot selection on Dashboard for live trading
- keep generic `/optimizer` focused on the existing OI bot flow
- move signal-bot-specific workflow into `/signal-bot` with 3 tabs:
  - Live Signals
  - Settings and Optimizer
  - Report

## 2) What changed

### 2.1 Optimizer page became embeddable and force-bot aware
File:
- `frontend/src/pages/optimizer/OptimizerPage.tsx`

Added props:
- `embedded?: boolean`
- `hideBotSelectors?: boolean`
- `forcedBotId?: string`
- `forcedBotPresetId?: string`

Behavior:
- loop start payload now uses forced bot/preset when provided
- in embedded mode:
  - header is hidden
  - bot selectors can be hidden
  - duplicate process-indicator strip is not rendered

### 2.2 Generic optimizer route locked to OI momentum bot
File:
- `frontend/src/app/routing/routes.tsx`

Route `/optimizer` now renders:
- `<OptimizerPage forcedBotId="oi-momentum-v1" hideBotSelectors />`

This removes cross-bot switching on the shared optimizer page and keeps signal-bot optimizer flow on its dedicated page.

### 2.3 Bot selector component supports constrained usage
File:
- `frontend/src/features/bots/components/BotExecutionSelectors.tsx`

Added props:
- `allowedBotIds?: string[]`
- `hideBotSelect?: boolean`

Behavior:
- can constrain the selector to a single bot and hide bot dropdown
- keeps preset selection visible
- aligns selected bot with allowed set

### 2.4 Signal bot page rewritten with 3 tabs
File:
- `frontend/src/pages/signalBot/SignalBotPage.tsx`

Implemented:
1. **Live Signals tab**
   - live table from WS rows
   - 10-second refresh cycle
   - Start tracking / Stop tracking toggle
   - basic live counters (total, long, short)
2. **Settings and Optimizer tab**
   - constrained bot preset selector for `signal-multi-factor-v1`
   - config form block
   - embedded optimizer forced to `signal-multi-factor-v1`
3. **Report tab**
   - summary cards
   - top symbols by absolute OI move

The page also auto-aligns selected bot to `signal-multi-factor-v1` for this dedicated workflow.
